# Code of Conduct

## Our Pledge

We pledge to make participation in this project respectful and harassment-free for everyone.

## Expected Behavior

- Be kind, direct, and constructive.
- Respect consent, privacy, and personal boundaries.
- Give and receive feedback in good faith.
- Use examples and test media that you have the right to use.
- Assume real people may be affected by how face-swap tools are presented and used.

## Unacceptable Behavior

- Harassment, insults, threats, or discriminatory language.
- Sharing private, intimate, or identifying media without permission.
- Encouraging impersonation, fraud, abuse, or non-consensual face swapping.
- Publishing another person's private information.
- Repeatedly derailing technical discussions after maintainers ask you to stop.

## Enforcement

Maintainers may remove comments, reject contributions, block users, or report abuse when behavior violates this code. Enforcement decisions should prioritize the safety and privacy of people affected by the project.

## Reporting

Report conduct concerns privately through the repository owner's GitHub profile or through a GitHub security advisory when the concern involves sensitive media or safety risk.
